YouTube-8M Video Understanding Challenge


Aimed to temporally localize the entities in the videos. Used various pooling methods to effectively aggregate the feature frames of a video. Took advantage of weak supervision information when training the models.
Won the Second place at the 3rd YouTube-8M Video Understanding Challenge of ICCV’19 Workshop.

Deepfakes Detection


Aimed to identify realistic AI-generated videos where people's faces have been replaced with fake faces by Deepfake techniques. Used these well-trained feature extractors and advanced pooling methods to build video classification models, which could identify fake videos.

Learn to Write Like You with a Few Examples


Student project, Google Machine Learning Winter Camp 2020.
The main goal of this project is to learn to write Chinese characters like you with only a few examples, which means users can create their own fonts with a small amount of handwriting Chinese characters.  This project presents a novel GAN framework, based on pixel2pixel algorithm and utilizing the transfer learning idea to generate new style without retraining.
